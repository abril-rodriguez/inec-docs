# Gastos

Un caso involucra un **gasto**, que conlleva el desembolso de dinero o la utilización de recursos para cubrir necesidades, adquirir bienes o servicios, o realizar actividades que generen valor o satisfagan requisitos específicos. Todos estos gastos pueden ser registrados en el sistema para llevar un control de ellos. Es importante aclarar que el gasto pertenece a un caso.{{ className: 'lead text-justify' }}

## Funcionalidades

Desde el módulo de gastos podemos:

1. Registrar un nuevo gasto.
2. Desde el listado de gastos podemos:

- Acceder a una vista rápida del gasto que permite ver toda su información.
- Editar los datos de un gasto.
- Eliminar un gasto.

---

## Nuevo gasto

Podemos crear un nuevo gasto desde el botón **registrar gasto**. El gasto puede estar relacionado a un cliente o a un caso.
