# Tareas

Una **tarea** se refiere a una actividad específica o un conjunto de acciones que se planifican y llevan a cabo con el fin de lograr un objetivo determinado dentro de un proyecto, proceso o sistema. Las tareas son unidades básicas de trabajo que deben completarse para avanzar hacia la finalización de un proyecto o para cumplir con un objetivo establecido.{{ className: 'lead text-justify' }}

En el sistema hay dos maneras de crear tareas: {{ className: 'lead text-justify' }}

1. Creando una desde el listado de tareas o desde la vista de un proyecto en particular. Esta será una tarea **independiente**.{{ className: 'lead text-justify' }}
2. Convirtiendo los procedimientos de un proceso a tareas de un proyecto. Esta será una **tarea de un proceso**.{{ className: 'lead text-justify' }}

## Funcionalidades

Desde el módulo de tareas podemos:

1. Registrar una nueva tarea.
2. Desde el listado de tareas podemos:

- Crear una tarea.
- Cambiar el estado de una tarea.
- Cambiar la prioridad de la tarea.
- Añadir un archivo a la tarea.
- Editar una tarea.
- Eliminar una tarea.
- Ver un listado de los cambios de la tarea.

---

## Nueva tarea

Una tarea siempre está relacionada con un proyecto de un colaborador.

Entre sus características fundamentales están el **estado** y las **dependencias**.
Su **estado** hace referencia a la situación en la que se encuentra dicha tarea. Los posibles estados son: sin iniciar, en proceso o culminada.{{ className: 'text-justify' }}

Las **dependencias** indican si esta tarea depende de otras o no. En el caso de que dependa de otra, hasta que dicha tarea no esté culminada, no se podrá cambiar el estado de la tarea actual.
Sumando a esto, algunas tareas pueden requerir un archivo para poder ser culminadas por lo que en caso de no presentar un archivo, la tarea se encontrará bloqueada.{{ className: 'text-justify' }}

## Cambiar el estado de una tarea

El estado puede ser cambiado desde la edición de tareas. Dicho estado no podrá ser cambiado, es decir la tarea se encuentra bloqueada, en el caso de que:

- La tarea requiera un archivo y aún no se haya subido.
- La tarea dependa de otra que aún no está culminada.

---

## Cambiar la prioridad de una tarea

La prioridad puede ser cambiada desde la edición de tareas. Las prioridades posibles en las que se puede encontrar una tarea son: bajo, medio, alto y urgente.{{ className: 'text-justify' }}
