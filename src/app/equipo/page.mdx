# Equipo

Desde aquí podemos ver y administrar a todos los miembros del staff que tienen acceso al sistema.{{ className: 'lead text-justify' }}

## Funcionalidades

Desde el módulo de equipo podemos:

1. Crear un nuevo miembro del equipo.
2. Ver una lista de todos los miembros del equipo.
3. Ver una vista detalle de cada miembro del equipo.

---

## Nuevo miembro

Desde el botón **crear nuevo miembro del equipo** podemos crear un nuevo miembro que tendrá acceso al sistema. Para que este usuario pueda ingresar, deberá registrarse con el correo con el que se lo dio de alta. {{className: "text-justify"}}

## Listado de miembros

El listado muestra todos los miembros del staff que tienen acceso al sistema y si se encuentran o no activos. Desde aquí podemos modificar si el usuario está activo o no.

<Image src="/images/equipo-1.png" alt="Equipo" width={1091} height={1003} />

## Vista detalle de un miembro

Haciendo click sobre el nombre del miembro del equipo podemos ver una vista detalle del mismo.
